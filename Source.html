Tags in html

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and user understandable content.

Some commonly used tags in HTML

    <html>
    It is the root element and contains all the other HTML elements in the document. It serves as the container for the entire HTML content.
    <head>
    It contains metadata, including the document title, character encoding, CSS and JavaScript links, and more.
    <body>
    It contains the visible content of the web page, such as text, images, links, and other HTML elements.

    The general syntax of any website is:

                      
<!DOCTYPE html>
    
<html>
    
      <head>
    
                              <!-- Meta tags, title, CSS links, etc. go here -->
    
      </head>
    
      <body>
    
            <!-- Content of the web page goes here -->
    
      </body>
    
</html>

    heading
    As the name suggests, it declares the heading for the page. It is divided into 6 styles, starting with h1 to h6. Where h1 beeing the biggest and h6 beeing smallest.
    <p>
    It represents a paragraph of text.
    <a>
    It creates a hyperlink to another web-page or resource.
    <img>
    It embeds an image to the web-page.
    <ul>
    It defines an unordered list.
    <ol>
    It defines an ordered list.
    <li>
    It is used inside the ul or ol tag.

    Syntax for a list: 

                      
<ul>
    
      <li>Item 1</li>
    
      <li>Item 2</li>
    
      <li>Item 3</li>
    
</ul>

    <div>
    It is a generic container used to apply css on a group of elements / tags.
    <span>
    It is a generic container used to apply css at a specific region withen a tag.
    <table>
    It is used to create a table.
    <tr>
    It defines a row in a table.
    <th>
    It defines a header cell in a table.
    <td>
    It defines a standard cell in a table.

    Syntax for a table tag:

                      
<table border="1">
    
      <tr>
    
            <th>Product Name</th>
    
            <th>Price</th>
    
            <th>Stock Quantity</th>
    
      </tr>
    
      <tbody>
    
            <tr>
    
                  <td>Product A</td>
    
                  <td>$25.99</td>
    
                  <td>150</td>
    
            </tr>
    
            <tr>
    
                  <td>Product B</td>
    
                  <td>$19.99</td>
    
                  <td>75</td>
    
            </tr>
    
      </tbody>
    
</table>

    Result:
    Product Name Price Stock Quantity
    Product A $25.99 150
    Product B $19.99 75
